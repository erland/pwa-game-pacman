# Basic 2D Game Template

    Minimal Phaser + Vite starter wired to `@SCOPE/pwa-game-2d-framework`.

    ## Setup

    ```bash
    node scripts/init.mjs --name=my-new-game --scope=YOUR_GH_USER_OR_ORG --framework=latest
    pnpm install
    pnpm dev
    ```

    Edit scenes under `src/scenes/*` and the Phaser config in `src/main.ts`.
